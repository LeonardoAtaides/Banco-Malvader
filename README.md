### Manual

1. **Configure o Backend**:
   ```bash
   cd backend
   npm install
   # Configure o arquivo .env com suas credenciais MySQL
   npm run dev
   ```

2. **Configure o Frontend**:
   ```bash
   cd frontend
   npm install
   npm run dev
   ```

3. **Backend**: 
  ```bash
 cd backend 
 npm run dev
 ```
**Frontend**: 
  ```bash   
cd frontend 
npm run dev
```
4. **Banco de dados**:
  ```bash
npx prisma db pull
npx prisma generate
```
