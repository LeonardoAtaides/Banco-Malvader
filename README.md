#  Banco Malvader

### Manual

1. **Configure o Backend**:
   ```bash
   cd backend
   npm install
   # Configure o arquivo .env com suas credenciais MySQL
   npm run dev
   ```

2. **Configure o Frontend**:
   ```bash
   cd frontend
   npm install
   npm run dev
   ```

3. **Backend:** Execute cd backend && npm run dev
   **Frontend:** Execute cd frontend && npm run dev
